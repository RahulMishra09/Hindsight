# Hindsight Frontend

**Status: Placeholder — scaffold arrives in Week 4.**

The frontend will be a React 18 + TypeScript + Tailwind CSS + shadcn/ui single-page application.

## Planned Features (v1.0)

- Browse and search incidents with semantic similarity ranking.
- Filter by severity, type, date range, and source.
- View individual incident detail with similar incidents panel.
- Dashboard with incident frequency and MTTR metrics.

Read-only in v1.0 — no write operations from the UI.

## Tech Stack

| Concern | Choice |
|---------|--------|
| Framework | React 18 |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Components | shadcn/ui |
| Build | Vite |
| API client | TanStack Query |

## Development (Week 4+)

```bash
cd frontend
npm install
npm run dev
```
